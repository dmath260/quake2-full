Stroggo Clicker: Cookie Clicker in Quake 2

How to play:

Download "gamex86.dll," "config.cfg," and the "Quake 2 Mod" shortcut from the Downloads folder

Open file explorer and find the local files for Quake 2 (on Steam, this is C:\Program Files (x86)\Steam\steamapps\common\Quake 2)

Create a new folder in this directory called "stroggoclicker" and put both gamex86.dll and config.cfg in the folder

Launch the shortcut, and it will immediately load the mod

When in the game, press F1 for a help/stats menu explaining the controls

Enjoy!
